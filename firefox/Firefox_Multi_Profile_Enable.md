# Firefox: Multi Profile Aktivieren

## Voraussetzungen

Öffne einen neuen Tab und rufe `about:config` auf.

## Einstellungen

Suche nach:

`browser.profiles.enabled`

und setze den Wert auf:

`true` (dopopelcklick)

## Anwendung

Nun gibt es im Anwendungs Menü ganz oben einen neuen Eintrag Profiel mit dem Namen von aktuellen Profil.

Im Untermenü gibt es die möglichkeit

- Ein anderes Profil öffnen

- Profile verwalten

## Nutzen

Es gibt fälle wo über Cross-Site-Scripting Daten auf andere Doamins übertragen werden. Mit diesem Konzept kann wichtiges von unwichtigem getrennt werden. Wie z.B. so:

- Shopping und täglicher gebrauch

- E-Banking

- Trash um beim suchen viele unbekannte Seiten anzuschauen.
